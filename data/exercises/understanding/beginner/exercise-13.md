## Understanding - Beginner, exercise 13

### Text

Consider the following Python function:

```python
def f(s):
    if len(s) > 3:
        return s[3:]
    else:
        return s[:3]
```

What is the result of the execution of `f("bod")`?

### Solution
`"bod"`
