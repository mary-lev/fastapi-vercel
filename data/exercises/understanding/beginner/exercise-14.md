## Understanding - Beginner, exercise 14

### Text

Consider the following Python function:

```python
def f(n):
    result = list()
    while n > 0:
        result.append(n)
        n = n -1
    return len(result)
```

What is the result of the execution of `f(3)`?

### Solution
`3`
