## Understanding - Beginner, exercise 17

### Text

Consider the following Python function:

```python
def f(x, y):
    if x <= 0 and y != 0:
        return x / y 
    else:
        return y / x
```

What is the result of the execution of `f(3, 0)`?

### Solution
`0`
